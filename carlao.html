Saiba Mais Sobre o Curso

A Formação UX PRO foi criada para quem deseja entrar na área de Experiência do Usuário de forma simples, prática e completa. O curso mostra como entender o usuário, identificar problemas, criar soluções e melhorar interfaces digitais.

O que é UX?

UX significa Experiência do Usuário. É a área que estuda como uma pessoa se sente ao usar um site, aplicativo ou sistema.

Pesquisa com Usuários

Você aprenderá a fazer pesquisas, entrevistas e análises para entender melhor as necessidades das pessoas.

Prototipagem

Testes de Usabilidade

Descubra se uma interface está fácil de usar e aprenda a melhorar a experiência do usuário.

Heurísticas de Nielsen

No curso, você aprenderá as 10 Heurísticas de Nielsen, que são princípios usados para avaliar se uma interface é simples, clara e fácil de utilizar.

✓ Visibilidade do status do sistema
✓ Correspondência entre o sistema e o mundo real
✓ Controle e liberdade para o usuário
✓ Consistência e padrões visuais
✓ Prevenção de erros
✓ Reconhecimento em vez de memorização
✓ Flexibilidade e eficiência de uso
✓ Design estético e minimalista
✓ Mensagens de erro claras
✓ Ajuda e documentação quando necessário

Modelo Duplo Diamante

Você também aprenderá o Modelo Duplo Diamante, uma metodologia usada para entender problemas e criar soluções melhores através de quatro etapas.

1. Descobrir

Nesta etapa, o objetivo é pesquisar, observar e entender o problema e os usuários envolvidos.

2. Definir

Depois da pesquisa, é hora de organizar as informações e escolher qual problema será resolvido.

3. Desenvolver

Aqui são criadas ideias, soluções, fluxos, telas e protótipos para resolver o problema.

4. Entregar

A solução é testada, ajustada e preparada para ser entregue ao usuário.
